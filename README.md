学会如何感受，进而如何表达，最终真正了解自己的途径。

#### 微光

微光是我展示和提升自己的一种途径。基本上，他是一个更好的我，赋予了我所期望的一些美好特质；然而他又不脱离实际，没有非凡的能力，只是尽力把能做的事做好，寻找自己的意义。
